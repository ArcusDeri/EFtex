EFtex Servo drive control/running-off control for warp beam running-of

EFtex Thread Winder FWI with automatic thread guide

  • Features

  • Control in a switch cabinet with one programmable logic servo controller per running-off unit. Each running-off unit is driven by a servo drive motor.
     

  • The running-off parameters are controlled and configured on a touch screen control panel. The current values such as main shaft rpm, number of windings, diameter and running-off rpm, etc. can be read on the control panel.
